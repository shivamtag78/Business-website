# Image Assets Instructions

Please place your attached images in this folder to be used by the website:

1. **Father's Photo/Director**: Name it `director.jpg` or modify the `TechnicalAuthority.jsx` to use your file's name. Currently it uses a placeholder.
2. **Construction Site Photos**: Rename them to `site1.jpg`, `site2.jpg`, `site3.jpg`, etc., or update the paths in `Hero.jsx` and `Portfolio.jsx` and `Services.jsx`.
3. **Logo**: Place `logo.png` here and update `Footer.jsx` if you want to use the image instead of text.
